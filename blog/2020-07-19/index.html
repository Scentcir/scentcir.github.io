Mac下Hugo + Github Pages搭建个人博客

Jul 19 2020

什么是Hugo#

Hugo是一个用 Go编写的静态网站生成器，2013由 Steve Francia 原创，自 v0.14 (2015年) 由 Bjørn Erik Pedersen 主力开发，并由全球各地的开发者和用户提交贡献。Hugo 一般只需几秒钟就能生成一个网站（每页少于 1 毫秒），被称为“世界上最快的网站构建框架”，也使 Hugo 大受欢迎，成为最热门的静态网站生成器之一，被广泛采用。

什么是GitHub Pages#

GitHub Pages是 GitHub提供的一个，于2008年推出。可以用于存放静态网页，包括博客、项目文档甚至整本书。

步骤#

  1. 安装Homebrew
  2. 安装Hugo
  3. 创建Blog
  4. 安装Hugo主题
  5. 部署到Github Pages

安装Homebrew#

什么是Homebrew#

Homebrew是一款自由及开放源代码的软件包管理系统，用以简化macOS系统上的软件安装过程。

如何安装Homebrew#

官网给出的安装方法：将以下命令粘贴到终端。 不过这个方法不适合国内的Mac用户，网络下载速度太慢。

/bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)”

解决办法要么科学上网，要么替换镜像源：将以下命令粘贴到终端。

cd "$(brew --repo)"
git remote set-url origin https://mirrors.ustc.edu.cn/brew.git

安装后，可以进行下一步。

安装Hugo#

安装Hugo，将以下命令粘贴到终端。

brew install hugo

检查安装成功，将以下命令粘贴到终端。

hugo version

创建Blog#

这里的Blog是在本地创建的文件夹，在终端里输入以下命令：

hugo new site mybolg #mybolg是文件夹名称，可替换

输入以下命令可以看到该文件夹有没有创建成功：

ls -l

输入以下命令可以进入该文件夹：

cd mybolg/

这样Bolg的站点就在本地生成了。

安装Hugo主题#

安装主题可以在Hugo官网中找寻，这里以Slim举例，在终端里输入以下命令：

cd themes
git clone https://github.com/zhe/hugo-theme-slim slim

下载好了，就可以在mybolg/themes里看到一个新的主题文件夹。 在终端输入以下命令，可以生成一个本地站点，将 http://localhost:1313/ 输入浏览器可预览：

hugo server -D

新建文章：

hugo new post/blog.md

部署到Github Pages#

首先你得有一个Github账号，如果没有的话请注册。 我在这里使用的是Github Desktop，GitHub Desktop可以不用gitShell就可以提交自己的代码工程，同时可视化的操作，让我觉得很方便。

安装GitHub Desktop#

GitHub Desktop

新建一个仓库（New Respository）#

这里新建仓库最好是在Github Desktop里面新建，这样远程和本地都会自动生成。仓库名命名必须是yourname.github.io，注意命名规范。

接着直接在终端（必须是根目录）里面执行：

hugo

此时，所有的静态页面都会生成到public目录里面。 将public目录里的文件全部复制粘贴到本地git仓库里面，在GitHub Desktop点击提交就可以了。
